= Inicio rápido para migrar a Google Cloud VMware Engine usando el asesor de migración de cargas de trabajo de VMware
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Ponte en marcha con el asesor de migración de VMware para trasladar tu infraestructura actual a Google Cloud VMware Engine utilizando Google Cloud NetApp Volumes como almacenes de datos NFS externos.

Antes de empezar, debes tener una idea de https://docs.netapp.com/us-en/workload-setup-admin/permissions-reference.html["permisos para NetApp Workload Factory"^].

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Una"]Iniciar sesión en Workload Factory
[role="quick-margin-para"]
Necesitarás https://docs.netapp.com/us-en/workload-setup-admin/sign-up-saas.html["Configurar una cuenta con Workload Factory"^] e inicie sesión usando uno de los https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html["experiencias de consola"^] .

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dos"]Agregar credenciales y permisos
[role="quick-margin-para"]
Elige el https://docs.netapp.com/us-en/workload-setup-admin/permissions-reference.html["políticas de permisos"] para satisfacer sus necesidades.

[role="quick-margin-para"]
Si decides no conceder permisos, puedes empezar a usar Workload Factory para VMware para copiar muestras de código parcialmente completadas.

[role="quick-margin-para"]
Si decides conceder permisos, tendrás que https://docs.netapp.com/us-en/workload-setup-admin/add-credentials.html["agregar credenciales a una cuenta manualmente"] que incluye proporcionar una cuenta de servicio de GCP con los permisos necesarios para ver y administrar los recursos de Google Cloud NetApp Volumes, la configuración de VPC y la conectividad del datastore de Google Cloud VMware Engine.

[role="quick-margin-para"]
https://docs.netapp.com/us-en/workload-setup-admin/add-credentials.html["Aprenda a agregar credenciales y permisos"^].

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Tres"]Cargar datos de inventario de VM
[role="quick-margin-para"]
Puede utilizar el centro de planificación para cargar detalles de inventario sobre su entorno de VM actual.  Utilice el script del recopilador de datos, RVTools o NetApp Data Infrastructure Insights para capturar la configuración actual de su máquina virtual y cargar los datos en el centro de planificación.

[role="quick-margin-para"]
link:upload-vm-inventory.html["Aprenda a cargar datos de inventario de máquinas virtuales"^] .

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Cuatro"] Crear un plan de puesta en marcha utilizando el asesor de migración de VMware
[role="quick-margin-para"]
Inicia el VMware migration advisor y configura el entorno de VM que quieres crear en la infraestructura de Google Cloud VMware Engine usando Google Cloud NetApp Volumes como datastores NFS externos.

[role="quick-margin-para"]
link:launch-migration-advisor-gcve.html["Descubre cómo utilizar el asesor de migración de VMware para crear un plan de puesta en marcha"].
